### 步驟

1. 先使用 **Adobe After Effect** 做好想要的動畫
2. 並且安裝好 **Bodymovin** 
3. 輸出成 ```data.json```
4. 放入sample的資料夾內
***

### 注意事項
單純用瀏覽器(Chrome)打開會不能讀取json，要使用其他瀏覽器，或直接丟到其他地方去host page。
